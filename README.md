<h1 align="center">Welcome node ts quickstart 👋</h1>

> This repository contain a quick start node/.ts

## Install

```sh
pnpm install
```

## Run build

```sh
pnpm build
```

## Run start

```sh
pnpm start
``` 

## Run tests

```sh
pnpm test
```
